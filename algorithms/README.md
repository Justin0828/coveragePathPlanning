# Algorithms

这里的三个原始算法模块是冻结实现：地图分割、邻接图/遍历顺序、矩形内往复式覆盖。此次仓库重构只移动了三个文件，没有修改其字节内容；`MANIFEST.json` 记录了 SHA-256，可由测试自动核验。

算法模块仍保留原型时期的同级导入方式。`src/algorithm_api.py` 是编排层与算法层之间唯一的导入边界，因此无需为了目录重构去改算法源码。

`segmentation_methods/` 是冻结实现之上的实验策略层，组合原有的 row/col 候选生成函数，提供 `row_only`、`col_only`、`row_then_col` 三种受控变量；它不复制或改写原始候选生成算法。
